An interactive Power BI dashboard that provides comprehensive insights into Netflix viewership patterns, user engagement, content preferences, and viewing trends. This project analyzes user behavior across multiple dimensions including demographics, devices, genres, and temporal patterns.

- **Most Active Age Groups**:

- **Cross-generational Appeal**: Strong adoption across all age segments, indicating Netflix's broad market penetration

### Geographic Insights
- **UK**: Highest average watch time (95 minutes) - strong binge-watching culture
- Other regions maintain consistent 91-94 minute averages

### Growth Trends
- **Peak Viewership**: August 2025 with 125,358 minutes
- **Consistent Growth**: Steady increase from January (121,437) through August
- **YoY Performance**: Positive month-over-month growth trajectory
